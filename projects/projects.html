Acquacchi

github play!

A strong chess engine written in C, ported to Javascript. It expands alpha-beta search with heuristics and bit-level manipulations. Acquacchi evaluates over 1.5 million positions every second.

ARC Dataset

github explore collected data

An interface to collect data about how people communicate and interprate descriptions of abstract patterns. This data will be used to inform a Bayesian program learner with the goal of solving the Abstraction and Reasoning Challenge--a benchmark for abstract thinking ability in programs developed by Google’s François Chollet.
