Acquacchi github play!

A chess engine of ELO ~2000, written in C. It expands alpha-beta search with heuristics and bit-level manipulations. Acquacchi evaluating over 1.5 million positions every second.

Kandula beta

API and app that enables anyone to create and customize stock-trading algorithms without code, just by tuning parameters. The program runs a plethora of trading algorithms on the cloud involving quantitative analysis, sentiment analysis, and GRU.

Dare beta

A social media app that revolves around completing “dares” instead of posting updates. The data is stored in NoSQL as a trial of how far non-relational data can be pushed.

ARC Dataset try it!

An interface to collect data about how people communicate and interprate descriptions of abstract patterns. This data will be used to inform a Bayesian program learner with the goal of solving the Abstraction and Reasoning Challenge--a benchmark for abstract thinking ability in programs developed by Google’s François Chollet.
