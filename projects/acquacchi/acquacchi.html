Acquacchi

Welcome to playing against Acquacchi!

This is a simple chess interface, where you have unlimited time and acquacchi has around 3 seconds per move. If you want to play against a stronger version (not compiled to web assembly), using any other time control, or using a much better interface, challenge Acquacchi on lichess.com. To send a challenge, you have to be logged in. Also, Acquacchi will only accept "Standard" and "From Position" variants with "Real time" time control.

Good luck!
